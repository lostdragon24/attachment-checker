# Attachment and Message Checker для Evolution

Плагин для почтового клиента Evolution, который проверяет письма на наличие запрещённых слов перед отправкой.

## Возможности

- Проверка имён файлов вложений
- Проверка текста письма
- Регистронезависимая проверка (опционально)
- Гибкие настройки через интерфейс Evolution
- Хранение настроек в текстовом файле
- Интернационализация

## Установка

```bash
make
sudo make install
